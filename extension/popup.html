Quick Start

1. Start the desktop app: python main.py

2. Select text on any web page

3. Right-click → "Read aloud"

Or click "Read Selection" in the Player tab

Requirements

The desktop app must be running for

this extension to work. It serves TTS

locally at localhost:7778.

Status dot shows READY when connected.

Right-Click Menu

Read aloud — plays through speakers

Read aloud & save WAV — plays + saves file

Stop reading — stops playback

Engines

Kokoro — local, free, no API key

OpenAI TTS — requires API key in config

ElevenLabs — requires API key in config

Preview plays a short sample for the selected voice.

Config: ~/.readout/config.json

Voices

18 built-in Kokoro voices

Blending: af_heart:60,am_adam:40

Change in Player tab or desktop app

Troubleshooting

OFFLINE — start the desktop app

No toast — reload the web page

No audio — check system sound output

Orange dot — model still loading

Won't work on chrome:// pages
